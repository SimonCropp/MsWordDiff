USAGE
  diffword <path1> <path2> [options]
  diffword [command] [...]

DESCRIPTION
  Compare two Word documents side by side using Microsoft Word

PARAMETERS
* path1             Path to the first Word document 
* path2             Path to the second Word document 

OPTIONS
  --quiet           Hide source documents in the comparison view. Default can be set via 'set-quiet' command 
  -h|--help         Shows help text. 
  --version         Shows version information. 

COMMANDS
  set-quiet         Set the default value for the Quiet option. Settings are saved in the user configuration file. 
  settings          Display the settings 

You can run `diffword [command] --help` to show help on a specific command.
